## 第2章 语音增强和深度学习理论基础

### 2.2.1 谱减法

谱减法是最经典的单通道语音增强算法之一，由 Boll 提出，Berouti 改进推广。

**基本假设**：噪声为加性噪声且与语音信号不相关。

**核心公式**：

> |Ŝ(t, f)| = |Y(t, f)| − |D̂(t, f)|　　　(2.2)

从带噪语音的短时幅度谱 |Y(t,f)| 中减去估计的噪声幅度谱 |D̂(t,f)|，得到纯净语音的短时幅度谱 |Ŝ(t,f)|。

**相位处理**：增强语音的时域信号通过估计的幅度谱 |Ŝ| 与**带噪语音的相位谱** ∠Y(t,f) 结合，经 iSTFT 得到：

> ŝ(n) = iSTFT( |Ŝ(t,f)| · e^{j∠Y(t,f)} )　　　(2.3)

**为什么用带噪相位？** 因为相位估计极其困难——人耳对相位不敏感，直接用带噪相位在大多数情况下足够。但这在低 SNR 时会导致明显的音质下降。

### 2.2.2 WPE算法

WPE (Weighted Prediction Error) 是一种基于长时线性预测的去混响方法，2008 年提出。

**基本思想**：从传声器拾取信号 Y(t,f) 中减去估计的晚期混响成分，获得直达声和早期反射声在最大似然意义上的最优估计。

**单通道 WPE**：

> Ŝ(t, f) = Y(t, f) − Σ_{k=1}^{P} W*(Δ+k, f) · Y(t−Δ−k, f)　　　(2.4)

即：Ŝ(t,f) = Y(t,f) − **W**ᴴ(f) **Y**(t,f)　　　(2.5) (向量形式)

其中：

| 符号 | 含义 |
|------|------|
| Δ | 预测时延（跳过早期反射的帧数，早期反射对语音清晰度有益） |
| P | 预测滤波器阶数（需要预测多少帧的混响） |
| W(Δ+k, f) | 频点 f 的第 k 个滤波器系数 |
| Y(t−Δ−k, f) | 当前帧 t 往前 Δ+k 帧的信号（过去的信号） |

**WPE 的迭代优化**：假设期望信号 S(t,f) 在每个时频点服从零均值、未知时变方差 λ(t,f) 的复高斯分布，通过交替迭代更新 λ(t,f) 和 W(f) 来最小化负对数似然函数 (2.6)。

```
迭代过程:
  1. 固定 W(f)，更新方差: λ̂(t,f) = |Ŝ(t,f)|²                    (2.7)
  2. 固定 λ(t,f)，更新滤波器: Ŵ(f) = (Σ Y(t,f)Yᴴ(t,f)/λ)⁻¹(Σ ...)   (2.8)
  3. 重复直到收敛
```

### 2.2.3 超指波束形成器

超指波束形成器 (Superdirective Beamformer) 是基于最大化指向性因子 (Directivity Index, DI) 的固定波束形成器。

**指向性因子 DI**：描述阵列对扩散噪声的抑制能力。

> DI(f) = |**w**ᴴ(f) **v**(f, θ_d)|² / (**w**ᴴ(f) **Γ**_nn(f) **w**(f))　　　(2.9)

| 符号 | 含义 |
|------|------|
| **v**(f, θ_d) | 导向向量（目标方向 θ_d） |
| **Γ**_nn(f) | 扩散噪声场的空间协方差矩阵 |
| **w**(f) | 波束形成器的权重向量 |

**优化问题**：在目标方向信号无失真约束 (**w**ᴴ**v** = 1) 下最大化 DI，等价于最小化输出扩散噪声功率。

> min **w**ᴴ **Γ**_nn **w**　　s.t. **w**ᴴ **v**(θ_d) = 1　　　(2.11)

**闭式解**：

> **w**(f) = **Γ**_nn⁻¹(f) **v**(f,θ_d) / (**v**ᴴ(f,θ_d) **Γ**_nn⁻¹(f) **v**(f,θ_d))　　　(2.12)

**特点**：
- 可以看作 MVDR 在扩散噪声场下的特例
- 对小孔径阵列，波束图在频率上有较好的一致性 → 适合宽带语音信号
- **白噪声增益 (WNG) 在低频段会变低甚至 < 1** → 对空间白噪声敏感 → 需要对角加载提高稳健性

---

### 2.2.4 MVDR波束形成器

MVDR (Minimum Variance Distortionless Response) 是自适应波束形成器。

**优化问题**：在保证期望方向信号无失真的条件下最小化输出噪声能量。

> min **w**ᴴ **R**_nn **w**　　s.t. **w**ᴴ **v**(θ_d) = 1　　　(2.13)

**闭式解**：

> **w**(f) = **R**_nn⁻¹(f) **v**(f,θ_d) / (**v**ᴴ(f,θ_d) **R**_nn⁻¹(f) **v**(f,θ_d))　　　(2.14)

**与超指波束形成器的区别**：

| 对比 | 超指 | MVDR |
|------|------|------|
| 噪声模型 | 理论扩散噪声场 **Γ**_nn | 实际估计的噪声协方差 **R**_nn |
| 自适应程度 | 固定（假设扩散场） | 自适应（根据实际输入调整） |
| 公式形式 | **Γ**_nn⁻¹ **v** / (**v**ᴴ **Γ**_nn⁻¹ **v**) | **R**_nn⁻¹ **v** / (**v**ᴴ **R**_nn⁻¹ **v**) |
| 需要的数据 | 阵列几何信息 | 噪声协方差矩阵（需在线估计） |
| 问题 | 对白噪声敏感、WNG低 | 导向向量/协方差不准→信号自消 |

### 2.3.1 编解码器结构

编解码器结构包括由卷积层堆叠的编码器和与之对称的解码器。

- **编码器**：通过卷积层从输入特征中提取高维特征（逐层压缩分辨率）
- **解码器**：通过卷积/反卷积层将低分辨率特征恢复到原始尺寸
- **跳转连接 (Skip Connection)**：编码层输出连接到对应解码层输入，促进深层梯度反向传播
- **因果约束**：时间维度的 padding 只能补过去一侧（因果填充），不能使用未来帧信息

这种结构天然适合输入输出尺寸相同的像素级修复任务（包括语音增强）。

---

### 2.3.3 时序卷积网络 TCN

TCN (Temporal Convolutional Network) 利用一维空洞卷积进行时序建模。

**结构**：若干 TCN 残差块堆叠，每个残差块包括：
- 时序建模分支：空洞因果卷积 → 归一化 → 激活函数 → Dropout
- 残差分支：1×1 点卷积

**空洞因果卷积 (Dilated Causal Convolution)** 的两个关键属性：

1. **因果 (Causal)**：卷积只能往过去看，padding 只在左侧补 0
2. **空洞 (Dilated)**：在卷积核的元素之间插入空隙（0），指数级扩大感受野

```
空洞率 d=1:  ● ● ●    感受野 = 3
空洞率 d=2:  ● ○ ● ○ ●    感受野 = 5（核大小不变，但隔一个位置取一个）
空洞率 d=4:  ● ○ ○ ○ ● ○ ○ ○ ●    感受野 = 9
空洞率 d=2^i: 感受野 = 2^i · (kernel_size−1) + 1
```

**TCN 相比 RNN 的优势**：
- 对长序列能**并行处理**（训练阶段可以看到整条序列）
- 感受野范围可手动设置
- 梯度稳定，避免 RNN 的梯度消失/爆炸
- 参数量更低

**TCN 的缺点**：
- 推理时需缓存感受野内的所有历史特征 → 占用更多存储
- **深层大空洞率会模糊短时细节**（卷积核被空洞拉得太稀疏，相邻帧信息被跳过）

### 2.4.1 训练目标：IRM 与 cIRM

#### IRM (理想比值掩蔽, Ideal Ratio Mask)

假设语音和噪声不相关，各时频点上语音能量的占比：

> M_r(t,f) = √[ |S(t,f)|² / (|S(t,f)|² + |D(t,f)|²) ]　　　(2.15)

- M_r ∈ [0, 1]
- M_r → 1：该时频点语音占主导
- M_r → 0：该时频点噪声占主导

**局限**：IRM 是实数掩蔽，只修正幅度谱，不处理相位。

#### cIRM (复数理想比值掩蔽, Complex Ideal Ratio Mask)

cIRM 同时处理实部和虚部，从而**间接修正相位**：

```
M_c(t,f) = M_r(t,f) + j·M_i(t,f)　　　(2.16)

其中：
  M_r = (Y_r·S_r + Y_i·S_i) / (Y_r² + Y_i²)   ← 实部掩蔽
  M_i = (Y_r·S_i − Y_i·S_r) / (Y_r² + Y_i²)   ← 虚部掩蔽
```

**cIRM 为什么能修正相位**：

```
Ŝ = M_c ⊙ Y  = (M_r+jM_i)(Y_r+jY_i)
   = (M_rY_r − M_iY_i) + j(M_rY_i + M_iY_r)

用复数乘法同时旋转角度（修正相位）和缩放模值（修正幅度）。
```

M_c 本身也是一个复数，其实部 M_r 和虚部 M_i 分别控制增强谱的实部和虚部。

### 2.4.2 基于信号近似的损失函数

**直接方法**：L = ‖M̂ − M‖²（计算掩蔽估计值和真实值的 MSE）

**信号近似 (Signal Approximation, SA)**：L = ‖M̂ ⊙ Y − S‖²（先重构谱再算 MSE）

**SA 的优势**：训练目标和最终听感目标一致——最终关心的是重构出来的**谱**准不准，而不是掩蔽本身准不准。SA 对掩蔽的微小估计误差更鲁棒。

---

### 2.4.3 幅度正则的复数谱损失函数

**核心问题**：相位谱没有清晰的结构，网络很难学习和重构。估计的相位偏差会导致网络"调偏幅度来补偿"，从而使幅度估计也变差。

**解决方案**：幅度正则的复数谱损失 = **复数谱损失 + 幅度谱损失**。

复数谱损失（Cartesian 形式）：
> L_RI = ‖Ŝ_real − S_real‖² + ‖Ŝ_imag − S_imag‖²

幅度正则项：
> L_Mag = ‖|Ŝ| − |S|‖²

带幂律压缩的变体（论文公式 2.18-2.20）：

对 STFT 谱做幂律压缩（β=0.3），再计算 MSE。压缩的作用：

| 压缩前（β=1） | 压缩后（β=0.3） |
|-------------|----------------|
| 强能量时频点主导损失（比如低频、共振峰） | 强弱能量相对均衡 |
| 弱能量时频点（如辅音、高频）几乎不被优化 | 弱能量也被"拉起来"，被更公平地对待 |
| 可懂度细节容易被忽略 | 可懂度关键细节更好地被保留 |

**压缩复数谱用 Cartesian 形式实现**（与 polar 形式 |S|^β·e^{j∠S} 等价）：

```
Ŝ_comp_real = Ŝ_real / |Ŝ|^0.7   ← 等价于 |Ŝ|^0.3·cos(∠Ŝ)，因为 cos(θ) = Ŝ_real/|Ŝ|
Ŝ_comp_imag = Ŝ_imag / |Ŝ|^0.7
```

其中 0.7 = 1 − β = 1 − 0.3。

## 第3章 基于目标干扰特性频域建模的单通道语音增强算法

### 3.2 问题定义：公式 (3.3) 和 (3.4)

论文将降噪（谱减法）和去混响（WPE）统一为两阶段形式。

**公式 (3.3)** 预测滤波器系数向量：

> **W**(f) = [ W(Δ+1, f),  W(Δ+2, f),  …,  W(Δ+P, f) ]ᵀ

**公式 (3.4)** 历史信号向量：

> **Y**(t, f) = [ Y(t−Δ−1, f),  Y(t−Δ−2, f),  …,  Y(t−Δ−P, f) ]ᵀ

**关键澄清**：W(f) 的第一个参数 **Δ+k 不是时间帧索引，而是滤波器抽头序号（lag index）**。

| 公式 | 第一个参数的含义 | 例子 |
|------|-----------------|------|
| W(Δ+k, f) | 抽头序号（第几个预测系数） | W(5, f) = "第 5 个抽头的系数" |
| Y(t−Δ−k, f) | 时间帧索引（第几帧的信号） | Y(t−5, f) = "t 往前数第 5 帧的信号" |

**具体例子**：Δ=4（跳过4帧早期反射），P=3（3阶滤波器），当前帧 t=100：

```
W 的抽头序号:    Δ+1=5      Δ+2=6      Δ+3=7
                  ↓           ↓           ↓
Y 的时间帧:    t-5=95      t-6=94      t-7=93
```

W 本身不随时间 t 变化——它是每个频点 f 固定的 P 阶滤波器，对所有帧 t 通用。

### 3.2 两阶段框架：公式 (3.5) 和 (3.6)

论文将降噪和去混响统一为两阶段通用框架：

> **(3.5)**  Ê = G₁( Y ; Θ₁ )　——　第一阶段：干扰估计
>
> **(3.6)**  Ŝ = G₂( Y, Ê ; Θ₂ )　——　第二阶段：语音恢复

| 符号 | 含义 |
|------|------|
| Y | 传声器拾取信号的 STFT 复数谱（带噪/带混响） |
| Ê | 第一阶段估计出的干扰成分（噪声幅度谱或混响成分） |
| Ŝ | 第二阶段估计出的增强语音复数谱 |
| G₁(·), G₂(·) | 第一、第二阶段的映射函数 |
| Θ₁, Θ₂ | 对应映射函数的参数 |

**为什么用 G₂ 做非线性映射而不是直接相减 Ŝ = Y − Ê？** 简单的减法容易导致某些时频点的值为负或产生音乐噪声等问题，非线性映射更鲁棒。

**传统方法 vs IESRN**：

| | 传统方法 | IESRN |
|---|---|---|
| G₁（干扰估计） | 谱减法/WPE（手工设计，无参数） | IE-Net（神经网络） |
| G₂（语音恢复） | 直接相减 Ŝ = Y − Ê | SR-Net（神经网络） |

### 3.3.1 IE-Net 与 SR-Net

#### IE-Net (Interference Estimation Network) — 干扰估计网络

**目的**：从带噪信号中估计干扰成分特征，并作为副产物估计语音幅度谱。

**结构**：

```
输入: |Y|（带噪幅度谱）
  ↓
编码器（5层 ConvGLU，核 2×3，步长 1×2）
  ↓
TFM（6个 LSTDCU）← 时序融合模块
  ↓
  ├──→ 解码器 A → 语音 IRM → ⊙|Y| → 语音幅度谱 |Ŝ_IE|
  └──→ 解码器 B → 干扰 IRM → ⊙|Y| → 干扰幅度谱 |Ê|
```

**设计要点**：

| 特点 | 说明 |
|------|------|
| 双解码器（多任务学习） | 同时估计语音和干扰的 IRM |
| 基于掩蔽 (mask-based) | 预测 0~1 的增益，不直接预测谱 |
| 容忍细节缺失 | 第一阶段只需找到干扰占主导的时频点 |

#### SR-Net (Speech Recovery Network) — 语音恢复网络

**目的**：利用 IE-Net 提供的干扰特征，从带噪信号中恢复干净语音复数谱。

**结构**：

```
输入: 复数谱 Y + IE-Net 估计的语音幅度谱 |Ŝ_IE|
  ↓
编码器（5层 ConvGLU）
  ↓                        ← ITM 多尺度干扰特征注入解码器各层
TFM（6个 LSTDCU）
  ↓
  ├──→ 解码器 A → 语音复数谱实部 Ŝ_R
  └──→ 解码器 B → 语音复数谱虚部 Ŝ_I
```

**设计要点**：

| 特点 | 说明 |
|------|------|
| 双解码器 | 分别估计实部和虚部 |
| 基于谱映射 (mapping-based) | 直接预测谱值，有强重构能力 |
| 接收干扰先验 | 通过 ITM 多尺度注入，告诉 SR-Net "哪里需要抑制" |

#### 两阶段设计的对比

| 对比维度 | IE-Net（阶段一） | SR-Net（阶段二） |
|----------|-----------------|-----------------|
| 目标 | 找到干扰在哪、是什么样 | 恢复干净的语音 |
| 方法 | 时频掩蔽（IRM） | 谱映射（直接预测复数谱） |
| 输出 | 干扰幅度谱 + 语音幅度谱 | 语音复数谱的实部和虚部 |
| 精度要求 | 粗估计即可 | 精细重构 |
| 输入 | 只用幅度谱 | 复数谱 + IE-Net 的先验信息 |

### 3.3.2 卷积门控线性单元 ConvGLU

ConvGLU 是对 GLU (Gating Linear Unit) 的二维卷积推广。论文中使用 ConvGLU 和 DeconvGLU 代替普通卷积/反卷积。

**公式** (3.7)：

> **f** = (W ∗ e + b_w) ⊙ σ(V ∗ e + b_v)

| 符号 | 含义 |
|------|------|
| e | 输入特征图 |
| W, V | 两个独立的卷积核 |
| ∗ | 卷积操作 |
| σ(·) | Sigmoid 函数，输出 ∈ (0, 1) |
| ⊙ | 哈达玛积（逐元素乘法） |

**计算流程**：

```
输入 e
  ├──→ W ∗ e + b_w → 线性特征 h（任意实数值）
  └──→ V ∗ e + b_v → σ(·) → 门控值 g（0~1）

输出 f = h ⊙ g

  门控值 g → 1：该位置的信息"通过"
  门控值 g → 0：该位置的信息"被阻断"
```

**ConvGLU vs 普通卷积**：

| 普通卷积 | ConvGLU |
|---------|---------|
| f = ReLU(W∗e+b) | f = (W∗e) ⊙ σ(V∗e) |
| 对所有输入一视同仁 | 网络自适应选择哪些位置值得关注 |
| 丢弃（ReLU 截断负值） | 平滑的门控，梯度流动更好 |

**ConvGLU vs DeconvGLU**：

| | ConvGLU | DeconvGLU |
|---|---|---|
| 基础操作 | 普通卷积 | 转置卷积 |
| 作用 | 压缩特征图（编码器） | 扩展特征图（解码器） |
| 公式结构 | 相同 | 相同，但 ∗ 是转置卷积 |

### 3.3.3 时序融合模块 TFM

TFM (Temporal Fusion Module) 替代常用的 TCM，解决 TCM 在建模长时依赖时模糊短时细节的问题。

#### 整体结构

```
输入 e₀  [B, 512, T, F]
  │
  ├──→ LSTDCU₁ → e₁ = e₀ + out₁（d_up=2, d_low=32 → 短+长）
  ├──→ LSTDCU₂ → e₂ = e₁ + out₂（d_up=4, d_low=16）
  ├──→ ...
  └──→ LSTDCU₆ → e₆ = e₅ + out₆（d_up=64, d_low=1 → 很长+短）
        │
        └──→ GFU 门控融合 → 加权求和 → f_TFM
```

#### LSTDCU (长短时密集交互单元)

每个 LSTDCU 包含三部分：

```
输入 e_q [B, 512, T, F]
  ↓
① 1×1 点卷积：512→64 通道压缩，降参数量　　　　　　(3.8)
  ↓
② DBDCB 双分支密集交互块（核心）　　　　　　　　　   (3.9)
   ├── 上分支：dilation = 2^i
   │    密集连接 + 门控
   └── 下分支：dilation = 2^(Q−i)
        密集连接 + 门控
   → 门控融合两分支输出
  ↓
③ 1×1 点卷积 + 残差：64→512，e_{q+1} = e_q + conv(f_q⁽²⁾)　　(3.10)
```

**互补空洞率**（Q=6）：

| LSTDCU | 上分支 d=2^i | 下分支 d=2^(6-i) | 上分支感受野 | 下分支感受野 |
|--------|------------|-----------------|------------|------------|
| i=1 | 2 | 32 | 5帧(短) | 65帧(**长**) |
| i=2 | 4 | 16 | 9帧 | 33帧 |
| i=3 | 8 | 8 | 17帧 | 17帧 |
| i=4 | 16 | 4 | 33帧 | 9帧 |
| i=5 | 32 | 2 | 65帧(**长**) | 5帧(短) |
| i=6 | 64 | 1 | 129帧(很长) | 3帧(很短) |

**规律**：上分支空洞率递增（2→64），下分支递减（32→1）。当上分支"远眺"（长时），下分支就在"细看"（短时），形成互补。

**密集连接**：每个分支内部，第 k 层卷积的输入 = 前面所有层的输出拼接，浅层细节永不丢失。

**门控机制**：两分支输出通过门控自适应融合：
> h_fused = σ(Conv(h_up)) ⊙ h_up + σ(Conv(h_low)) ⊙ h_low

平稳段→长时分支权重高；突发噪声→短时分支权重高。

#### GFU (门控融合单元)

6 个 LSTDCU 的输出 {e₀, e₁, ..., e₆} 通过 GFU 预测每个 e_q 的权重图 A_q：

> f_TFM = Σ_{q=0}^Q A_q ⊙ e_q　　　(3.12)

A_q 与 e_q 同形状，值 ∈ (0,1)，对不同 (t,f) 位置自适应加权。

**TFM vs TCM**：

| TCM | TFM |
|-----|-----|
| 串行堆叠，深层丢失浅层细节 | 残差连接保证每层信息直达输出 |
| 每层只有一个空洞率 | 每层两个互补空洞率 |
| 多尺度靠层间隐式融合 | GFU 显式学习每个尺度权重 |
| 对突发噪声不敏感 | 短时分支专门捕获突发模式 |

### 3.3.4 干扰转换模块 ITM

ITM (Interference Transform Module) 的作用：将 IE-Net 输出的干扰幅度谱转换成多尺度的嵌入式特征，注入 SR-Net 解码器的各层。

**结构**：U-Net + FAM (Feature Attention Module) 级联。

- **U-Net**：对干扰特征进行变换，每个解码层输出不同尺度的干扰嵌入特征
- **FAM**：通过空间注意力和通道注意力对特征重新加权

ITM 的各层输出通过跳转连接注入 SR-Net 解码器的对应层，让 SR-Net 在不同分辨率下都能感知干扰的存在。

### 3.3.5 损失函数

#### 阶段一：只训 IE-Net

> L_IE = 0.5 × (‖M̂_s ⊙ |Y| − |S|‖² + ‖M̂_n ⊙ |Y| − |N|‖²)　　　(3.16)

- M̂_s, M̂_n：IE-Net 估计的语音/干扰 IRM
- 用**信号近似 (SA)** 方式：算重构谱的 MSE，而非掩蔽值的 MSE
- 0.5 是两项取平均的系数
- 两个任务一起训 → 多任务学习，同时理解"什么是语音"和"什么是干扰"

#### 阶段二：联合训练 IE-Net + SR-Net

> L_joint = λ · L_IE + α · L_SR_RI + (1−α) · L_SR_Mag　　　(3.17)

其中 λ=0.1，α=0.5。

**第二项：复数谱损失**（3.18）：

> L_SR_RI = ‖Ŝ_R − S_R‖² + ‖Ŝ_I − S_I‖²

SR-Net 输出实部和虚部与纯净谱的 MSE。

**第三项：幅度正则损失**（3.19）：

> L_SR_Mag = ‖|Ŝ| − |S|‖²

只算幅度 MSE，不管相位。

**幅度正则的核心逻辑**：

```
纯复数损失:
  网络相位偏了30° → 为了最小化复数误差调偏幅度来补偿
  → 幅度也变差了

加幅度正则:
  复数损失要调幅度 → 幅度正则不允许
  → 网络只能同时认真学相位和幅度
```

- **α = 0.5**：复数损失和幅度正则等权重，经验值
- **λ = 0.1**：阶段二 IE-Net 只需微调，主力优化 SR-Net
- 阶段一 IE-Net 学习率 0.001，阶段二降为 0.0001（防止被 SR-Net 梯度冲走）

## 补充：PESQ 和 ESTOI 指标

### PESQ (Perceptual Evaluation of Speech Quality)

**评估音质好坏**（好不好听），ITU-T P.862 标准。

**计算流程**：

```
增强语音 ŝ(n) / 纯净语音 s(n)
  ↓
① 电平对齐 → 统一到标准听音声压级
  ↓
② IRS 滤波 → 模拟电话听筒频率响应 (300Hz~3.1kHz)
  ↓
③ 时间对齐 → 补偿延迟偏差
  ↓
④ 听觉变换 → FFT → Bark 频带映射 → 响度计算
  ↓
⑤ 认知模型 → 对称干扰差（多出来的噪声）+ 非对称干扰差（缺失的语音）
  ↓
⑥ 时间平均 → 对各帧干扰差加权平均
  ↓
PESQ 分数 [-0.5, 4.5]
```

**关键机制**：

| 概念 | 说明 |
|------|------|
| Bark 频带 | 模拟人耳的非线性频率分辨率（低频密集、高频稀疏） |
| 响度映射 | 人耳对声强是对数感知（Sone 单位），不是线性 |
| 对称干扰 | 增强信号比纯净信号**多**出来的（噪声） |
| 非对称干扰 | 增强信号比纯净信号**缺**失的（语音失真），惩罚更重 |

**分数范围**：

| PESQ | 含义 |
|------|------|
| 4.5 | 极好，几乎听不出差别 |
| 2.5~3.0 | 一般，明显但可容忍 |
| 1.5~2.0 | 带噪语音的典型范围 |
| <1.0 | 极差 |

### ESTOI (Extended Short-Time Objective Intelligibility)

**评估可懂度**（听不听得懂），STOI 的改进版。

**计算流程**：

```
增强语音 ŝ(n) / 纯净语音 s(n)
  ↓
① STFT → 合并为 15 个 1/3 倍频程频带
  ↓
② 分块：每块 384ms (≈30 帧)
  ↓
③ 对每个时间块：
   - 取高能量帧（前 50%）
   - 归一化: Ŝ' = Ŝ/‖Ŝ‖, S' = S/‖S‖
   - 计算皮尔逊相关系数 r_k
  ↓
ESTOI = 所有块 r_k 的平均，[0, 1]
```

**归一化的意义**：对总体增益变化不敏感——即使增强语音音量偏大/偏小，只要频谱形状（包络）保持得好，分数不变。

### PESQ vs ESTOI

| 对比维度 | PESQ | ESTOI |
|---------|------|-------|
| 评估什么 | 音质（好不好听） | 可懂度（听不听得懂） |
| 听觉模型 | 有（Bark, 响度） | 无 |
| 对增益敏感 | 敏感 | 不敏感 |
| 分数范围 | -0.5 ~ 4.5 | 0 ~ 1 |
| 计算复杂度 | 较高 | 较低 |

## 补充：GTCRN 损失函数与论文公式的对应关系

GTCRN 的 [loss.py](loss.py) 实现了论文公式 (2.18-2.20) 的变体。

### 论文公式 (2.20) 的 Polar 形式

论文中压缩复数谱用极坐标表示：

> |Ŝ|^β · e^{j∠Ŝ}

其中 β = 0.3（压缩因子），|Ŝ|^0.3 是压缩后的幅度，e^{j∠Ŝ} 保留原始相位。

### loss.py 的 Cartesian 实现

```python
real_loss = MSE(Ŝ_real / |Ŝ|^0.7,  S_real / |S|^0.7)
imag_loss = MSE(Ŝ_imag / |Ŝ|^0.7,  S_imag / |S|^0.7)
mag_loss  = MSE(|Ŝ|^0.3,          |S|^0.3)
```

**等价性证明**：

```
极坐标形式: |Ŝ|^β · e^{j∠Ŝ} = |Ŝ|^β · cos(∠Ŝ) + j·|Ŝ|^β · sin(∠Ŝ)

因为 cos(∠Ŝ) = Ŝ_real/|Ŝ|,  sin(∠Ŝ) = Ŝ_imag/|Ŝ|

所以: |Ŝ|^β · cos(∠Ŝ) = |Ŝ|^β · Ŝ_real/|Ŝ| = Ŝ_real / |Ŝ|^{1−β} = Ŝ_real / |Ŝ|^0.7
     |Ŝ|^β · sin(∠Ŝ) = |Ŝ|^β · Ŝ_imag/|Ŝ| = Ŝ_imag / |Ŝ|^{1−β} = Ŝ_imag / |Ŝ|^0.7
```

### loss.py 总损失

```python
return 30*(real_loss + imag_loss) + 70*mag_loss + sisnr
```

| 项 | 有效权重 | 占比 | 对应论文 |
|----|---------|------|---------|
| 压缩复数损失 | 30+30=60 | ≈46% | α·L_RI |
| 压缩幅度损失 | 70 | ≈54% | (1−α)·L_Mag |
| SI-SNR | 1 | ≈1% | 额外时域项 |

**权重 30 和 70 是超参数**：由实验调参得出，不是数学推导。大原则是幅度略重于复数（53% vs 46%）——幅度关乎可懂度，优先级更高；复数损失保留相位信息，避免音质劣化。

**幂律压缩 β=0.3** 的作用：压缩大能量和小能量之间的差距，让低能量时频点（如辅音、高频细节）在损失中获得更公平的对待，提升可懂度。
